# 🚀 ProCBT

ProCBT is a modern Computer-Based Testing (CBT) platform designed for schools, universities, training centers, and organizations to conduct secure online examinations.

This project is being built from scratch using modern web technologies with the goal of providing a fast, secure, and user-friendly examination system.

---

## ✨ Features

### Student
- Register an account
- Secure login
- Take online exams
- View exam history
- Check results
- Update profile

### Administrator
- Secure admin login
- Manage students
- Create exams
- Manage question banks
- View exam reports
- Publish results
